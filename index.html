Kurulum gerekli

Uygulamanın çalışması için bir Firebase projesi bağlaman gerekiyor. Yaklaşık 5 dakika sürer, ücretsiz plandaki (Spark) limitler bu kullanım için fazlasıyla yeterli.

  1. console.firebase.google.com adresinde yeni bir proje oluştur.
  2. Authentication → Sign-in method altında E-posta/Şifre yöntemini aç. Users sekmesinden avukatın e-postasıyla bir kullanıcı ekle (uygulamada kayıt ekranı yok; kullanıcıyı buradan sen tanımlıyorsun).
  3. Firestore Database → Veritabanı oluştur de (production modu), ardından Rules sekmesine aşağıdaki kuralları yapıştırıp yayınla: 
    rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{koleksiyon}/{belge} {
      allow read, update, delete: if request.auth != null
        && request.auth.uid == resource.data.uid;
      allow create: if request.auth != null
        && request.auth.uid == request.resource.data.uid;
    }
  }
}
  4. Proje ayarlarında </> Web uygulaması ekle ve verilen firebaseConfig değerlerini kopyala.
  5. Bu HTML dosyasını bir metin düzenleyiciyle aç, en alttaki FIREBASE_CONFIG bölümüne değerleri yapıştır, kaydet.
