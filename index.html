Kurulum gerekli

Uygulamanın çalışması için bir Firebase projesi bağlaman gerekiyor.

  1. console.firebase.google.com adresinde yeni bir proje oluştur.
  2. Authentication → Sign-in method altında E-posta/Şifre yöntemini aç. Users sekmesinden kullanıcı ekle (uygulamada kayıt ekranı yok).
  3. Firestore Database → Veritabanı oluştur de, Rules sekmesine şu kuralları yapıştırıp yayınla: 
    rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{koleksiyon}/{belge} {
      allow read, update, delete: if request.auth != null
        && request.auth.uid == resource.data.uid;
      allow create: if request.auth != null
        && request.auth.uid == request.resource.data.uid;
    }
  }
}
  4. Proje ayarlarından </> Web uygulaması ekle, verilen firebaseConfig bloğunu kopyala.
  5. Bu HTML dosyasında YAPILANDIRMA BURADA yazan bölüme bloğu olduğu gibi yapıştır — değişken adı firebaseConfig ya da FIREBASE_CONFIG olabilir, ikisi de çalışır.
  6. GitHub Pages kullanıyorsan: Firebase Console → Authentication → Settings → Authorized domains listesine kullaniciadin.github.io ekle.
Yapılandırma henüz girilmemiş görünüyor.
